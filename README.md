# HTML Resume Website

This project is a single-page resume website built using **pure HTML**.  
It follows semantic HTML structure and matches the provided layout.

## Features
- Clean single-page resume
- Semantic HTML tags
- Sections for About, Skills, Experience, Projects, Education, and Contact
- Tables used for structured data
- No CSS or JavaScript used

## Technologies Used
- HTML5

## Setup Instructions
1. Clone the repository 
2. Open `resume.html` in any web browser

## Usage
This project can be used as:
- College assignment
- HTML practice project
- Beginner resume website
